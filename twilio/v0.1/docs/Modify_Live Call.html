POST modifyLiveCall

Modifies a live call. Modification could be redirection or termination of the call.

A valid URL that returns TwiML. Twilio will immediately redirect the call to the new TwiML.

Url

string

The HTTP method Twilio should use when requesting the above URL. Defaults to POST.

Method

Required

string

Either canceled or completed. Specifying canceled will attempt to hangup calls that are queued or ringing but not affect calls already in progress. Specifying completed will attempt to hang up a call even if it's already in progress.
