GETactivities.getUserActivities

Retrieves all activities for the user or for the friends of the viewer.

Test console
Description
Parameter
string

The person's identifier.

personId

Required

string

Indicates which set of individuals to query for activities. @friends and @all currently can be used interchangeably.

selector

Required

string

Only returns the nearest multiple of 3 compared to the original value.

count

Optional

string

The following field names are supported.
appid, body, externalid, id, mediaitems, postedtime, streamfaviconurl, title, titleid, userid,
@all (a special value that tells the REST endpoint to send back all fields for the activity.)

fields

Optional

string

Indicates the index of the first item to retrieve from the query set.

startIndex

Optional

string

Indicates the date before which no activities should be returned.

updatedSince

Optional
