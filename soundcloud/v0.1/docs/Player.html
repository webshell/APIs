Soundcloud Player Widget

Soundcloud.player allows you to embed a classic soundcloud widget

Method Description Exemple
player(args) Embeds the widget, the list of arguments can be found bellow. You can either call this method with the url or with an object containing other arguments(including the url) 
apis.soundcloud.player('http://soundcloud.com/matas/hobnotropic')
or
apis.soundcloud.player({default_height: 50, default_width: 450, url: 'http://soundcloud.com/matas/hobnotropic'})

Here is the list of arguments that can be passed to the apis.soundcloud.player funcion:

Argument type Description
color hex code Color play button and other controls. e.g. “0066CC”
theme_color hex code Color player background etc.
text_buy_track String e.g. “Tweet” and it points to your “buy” link
text_buy_set String e.g. “Tweet” and it points to your “buy” link
text_download_track String e.g. “Get it” and it points to your “download” link
buying true/false Show/Hide buy buttons
sharing true/false Show/Hide share buttons
download true/false Show/Hide download buttons
show_bpm true/false Show/Hide bpm display in player
show_playcount true/false Show/Hide number of track plays
start_track Number A number from 0 to the playlist length. preselects a track in a playlist
default_width Number A number e.g. 400. Overrides the player’s autoscale, useful when embedding in other flash movies
default_height Number A number e.g. 500. Overrides the player’s autoscale, useful when embedding in other flash movies
font String A string e.g. “Arial”. Overrides the default font with system font
enable_api true/false Enable the JavaScript API callbacks
single_active true/false if set to false the multiple players on the page won’t toggle each other off when playing
show_user true/false Show or hide the uploader name, useful e.g. in tiny players to save space)
auto_play true/false
show_artwork true/false
show_playcount true/false
show_bpm true/false
show_comments true/false

Here is the list of functions that can be used on a player object:

Method Description Example
play() Player starts playing either from 0 or from the last paused track position. In the case the widget contains multiple tracks it will start playing the first track. 
var player = apis.soundcloud.player("http://soundcloud.com/airin88/sets/metallica/"); player.play();
pause() Player stops playing, current track position is saved. 
var player = apis.soundcloud.player("http://soundcloud.com/airin88/sets/metallica/"); player.play();player.pause();
stop() Player stops playing, current track position is reset to 0. 
var player = apis.soundcloud.player("http://soundcloud.com/airin88/sets/metallica/"); player.play();player.stop();
toggle() Player starts or stops playing, depending on it's current state. The current track position is saved. 
var player = apis.soundcloud.player("http://soundcloud.com/airin88/sets/metallica/"); player.play();player.toggle();
skip(trackIndex) In the case the widget contains multiple tracks it will jump to the trackIndex, where the track list goes 0,1,2. e.g. player.skip(2) will load the third track in a playlist. You can check the current trackIndex by calling player.getCurrentTrackIndex(). 
var player = apis.soundcloud.player("http://soundcloud.com/airin88/sets/metallica/"); player.play();player.skip(1);
seekTo(seconds) Jump to a certain position in a track. Before seeking, please make sure that the track is fully buffered. 
var player = apis.soundcloud.player("http://soundcloud.com/airin88/sets/metallica/"); player.play();player.seekTo(50);
load(url) Player stops playing, resets and loads a new resource from the given url. This way one widget could be used as an audio engine for the multiple SoundCloud resources. 
var player = apis.soundcloud.player("http://soundcloud.com/airin88/sets/metallica/"); player.play();player.load(50);
next() In the case the widget contains multiple tracks it will skip to the next track. 
var player = apis.soundcloud.player("http://soundcloud.com/airin88/sets/metallica/"); player.play();player.next(50);
prev() In the case the widget contains multiple tracks it will skip to the previous track. 
var player = apis.soundcloud.player("http://soundcloud.com/airin88/sets/metallica/"); player.play();player.prev(50);
setVolume(value) Set the widget volume to a certain value. 
var player = apis.soundcloud.player("http://soundcloud.com/airin88/sets/metallica/"); player.play();player.setVolume(50);
getVolume() Returns current volume, as a number between 0 and 100. 
var player = apis.soundcloud.player("http://soundcloud.com/airin88/sets/metallica/"); player.play();var volume = player.getVolume();
getTrackDuration() Returns current track duration in seconds. 
var player = apis.soundcloud.player("http://soundcloud.com/airin88/sets/metallica/"); player.play();var duration = player.getTrackDuration();
getTrackPosition() Returns current track position in seconds. 
var player = apis.soundcloud.player("http://soundcloud.com/airin88/sets/metallica/"); player.play();var position = player.getTrackPosition();
/td>
getTracks() Returns the list of the track objects in the player 
var player = apis.soundcloud.player("http://soundcloud.com/airin88/sets/metallica/"); player.play();var tracks = player.getTracks();
getCurrentTrack() Returns the current track object 
var player = apis.soundcloud.player("http://soundcloud.com/airin88/sets/metallica/"); player.play();var track = player.getCurrentTrack();
getCurrentTrackIndex() Returns the index of the current track in the playlist (0 based). If the first track is playing it'll return 0, the second 1, and so on. 
var player = apis.soundcloud.player("http://soundcloud.com/airin88/sets/metallica/"); player.play();var trackIndex = player.getCurrentTrackIndex();
getFlashId() Returns the id/name of the widget node in the DOM. 
var player = apis.soundcloud.player("http://soundcloud.com/airin88/sets/metallica/"); player.play();var flashId = player.getFlashId();
